File tree
=========
 /src/ contains all the source files
 /doc/ contains the project documentation
 /bin/ contains socle-j2ee-core.jar result of the build
 /bin/classes/ contains all class and resource files from the build
 /bin/javadoc/ contains the api javadoc
 /cfg/ contains configuration files
 /cfg/eclipse.formater.xml is the eclipse java format prefs export
 /cfg/eclipse.preferences.epf is the eclipse general prefs export
 /lib/ contains all the jar files used by this project
 /make/ contains the ant and eclipse scripts
