Provides access generic interfaces to a legacy system.

General use of the APIs:

Here is a code sample for CICS Transaction Gateway implementation:

...

Properties props = new Properties();

props.setProperty(CtgServer.SOCLE_CTG_JNDI, "CICSConnectionFactory");

CtgServer server = new CtgServer(props);

//open the session

LegacySession session = server.openSession();

//ask for a new call

LegacyRequest interaction = session.newRequest();

interaction.setFunctionName("MEVAPI");

interaction.setInteractionVerb(InteractionSpec.SYNC_SEND_RECEIVE);

interaction.setTPNName("INCS");

interaction.setTranName("INCS");

...

Once the LegacyRequest got back, the values must be written in order to create the commarea itself. See javadoc of LegacyServer or LegacyResponse for more details.
