TOOLS Revision History

Version 1.4.6

New features

  • vogen - highway.mapping.property annotation has new "insert" and "update" parameters (optional).

Dependencies

  • TOOLS 1.4.6 must be used in conjunction with CORE 1.4.7.

Version 1.4.5

New features

  • Evo #125 : serialVersionUID may be added in generated value objects through a new annotation : highway.vo.serialVersionUID.

Dependencies

  • TOOLS 1.4.5 must be used in conjunction with CORE 1.4.6.

Version 1.4.4

Defects fixed

  • Defect #122 : Wrong generation proxy method name using EJB compression.

Dependencies

  • TOOLS 1.4.4 must be used in conjunction with CORE 1.4.5.

Version 1.4.3

New features

  • Evo #88 : highway.mapping.property and highway.mapping.id annotations have a new "type" parameter (optional), which specifies the Hibernate type of the property.

Dependencies

  • TOOLS 1.4.3 must be used in conjunction with CORE 1.4.3.

Version 1.4.2

New features

  • EJB service compression : ejbgen can now generate ejb services that compress service parameter and return value. This is a final implementation. You can use it on project.

Dependencies

  • TOOLS 1.4.2 must be used in conjunction with CORE 1.4.2.

Version 1.4.1

Defects fixed

  • Defect #71 : vogen now generates array properties.
  • Defect #75 : metagen now handles array arguments in methods.

New features

  • EJB service compression : ejbgen can now generate ejb services that compress service parameter and return value. This is a draft implementation. Use it only for tests.

Dependencies

  • TOOLS 1.4.1 must be used in conjunction with CORE 1.4.1.
