File tree
=========
 /src/ contains all the source files needed to build socle-j2ee-tools.jar
 /src/java/ contains the java source files and some of the resource files (*.xdt)
 /src/lib/ contains the jar files that are merged in socle-j2ee-tools.jar
 /src/cfg/ contains modified resource files overriding those contained in src/lib/*.jar files
 /make/ contains all the ant (*.build.xml) and eclipse (*.launch) build scripts
 /bin/ contains socle-j2ee-tools.jar result of the build
 /bin/classes/ contains all class and resource files from the buid
 /lib/ contains jar files needed for compilation
 /lib/ant.jar is version 1.6.2 and is needed for ant task compilation
