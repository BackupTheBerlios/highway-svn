
The socle tools need commons-attribute 2.1.
List of included files:
- commons-attributes-api-2.1.jar
- commons-attributes-compiler-2.1.jar
- commons-collections-2.1.jar
- xjavadoc-1.0.jar (replaced by version 1.1 from xdoclet)
 
The socle tools needs xdoclet.
List of included files:
- xdoclet-1.2.2.jar
- xjavadoc-1.1.jar
- commons-logging.jar
- commons-collections-2.0.jar (replaced by version 2.1 from commons-attribute)
- xdoclet-ejb-module-1.2.2.jar (for task ejbdoclet)
- xdoclet-ibm-module-1.2.2.jar (for subtask websphere)
- xdoclet-web-module-1.2.2.jar (for subtask web needed by subtask websphere)
 
Ejbdoclet needs the j2ee classes to work.
We have included the file j2ee.jar from WebSphere 5.0.2.4.

The socle tools need some of commons-lang functionalities.
List of included files:
- commons-lang-2.0.jar
